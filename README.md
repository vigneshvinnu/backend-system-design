# Backend System Designs

This repository contains system design documents for scalable backend services.
Each design focuses on real-world constraints such as scalability, reliability,
failure handling, and trade-offs.

## Covered Topics
- Rate limiting and traffic control
- Asynchronous notification systems
- Distributed job scheduling
- Search systems with caching

All designs include:
- Requirements and assumptions
- High-level architecture
- Data models
- API contracts
- Scaling strategies
- Failure scenarios and trade-offs
